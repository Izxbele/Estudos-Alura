# Análise de Dados com Pandas: Aplicações e Desafios Práticos

Este repositório contém notebooks desenvolvidos com foco na manipulação e análise de dados tabulares utilizando a biblioteca **Pandas**, ferramenta fundamental para a ciência de dados em Python. Os materiais foram elaborados com fins acadêmicos e educacionais.

## Estrutura do Repositório

- `aula pandas_T.ipynb`: Notebook introdutório com conteúdos básicos de manipulação de dados utilizando Pandas. Serve como base teórica e prática para o desenvolvimento de projetos posteriores.
- `Desafios(t_m).ipynb`: Projeto aplicado com o título "Desafio 1 - Vendas Online", que propõe uma análise exploratória e estatística de dados comerciais, reforçando a utilização das funcionalidades do Pandas em um contexto realista.

## Objetivos

- Desenvolver fluência no uso da biblioteca Pandas para análise de dados.
- Praticar tarefas comuns em ciência de dados, como:
  - Leitura de arquivos CSV
  - Tratamento de dados ausentes
  - Cálculo de estatísticas descritivas
  - Agrupamentos e filtragens com `groupby()`
  - Criação de colunas derivadas
  - Análises específicas sobre desempenho de vendas

## Conjunto de Dados

O projeto "Vendas Online" utiliza um dataset simulado de transações comerciais em ambiente digital. Informações mais detalhadas sobre o conjunto de dados (formato, colunas, origem) estão documentadas diretamente no notebook `Desafios(t_m).ipynb`.
